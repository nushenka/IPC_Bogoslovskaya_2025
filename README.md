## Краткое описание

Данный проект представляет собой **многоагентную экономическую игру-симулятор**, в которой игроки управляют различными виртуальными бизнесами в условиях динамически меняющегося рынка. Модель имитирует ключевые макроэкономические процессы: инфляцию, изменение ставки рефинансирования, государственные налоги и субсидии, а также случайные рыночные шоки, влияющие на спрос и цены активов.

## Цель проекта

Разработать и проанализировать многоагентную модель, которая демонстрирует:
* Взаимодействие различных секторов экономики.
* Влияние макроэкономических факторов и случайных событий на микроэкономические решения агентов.
* Формирование стратегического поведения игроков на основе анализа данных (графиков спроса, котировок акций, новостной ленты).

## Функциональные возможности

* **Управление бизнесом**: Игроки могут покупать и управлять одним из нескольких типов компаний.
* **Динамическая экономика**:
  * **Инфляция**: Учитывается каждый раунд.
  * **Ставка рефинансирования**: Динамически меняется, влияя на стоимость кредитов.
  * **Налоги и субсидии**: Устанавливаются "правительством" (системой).
  * **Мировой спрос**: Влияет на уровень цен и спрос.
* **Случайные события (Шоки)**: Непредсказуемые события, которые кардинально меняют рыночную конъюнктуру.
* **Система спроса**:
  * Первые 10 раундов спрос генерируется случайно для "разогрева" модели.
  * После 10-го раунда спрос рассчитывается с помощью **линейной регрессии**, на которую влияют цена игрока, цены конкурентов, реклама и шоки.
* **Мини-аналитика**: Каждый игрок имеет доступ к графикам спроса на свою продукцию, истории прибыли и изменения цен на активы.
* **Фондовый рынок**:
  * **Биржа**: Покупка и продажа акций (долей) компаний.
  * **Драгметаллы**: Инвестирование в золото и другие металлы, цена которых волатильна.
* **Нелинейные функции издержек**: Для каждого типа бизнеса реализована своя модель затрат.

## Архитектура и агенты

В симуляторе представлены следующие типы агентов-компаний:

1.  **Строительная компания**
2.  **Нефтедобывающая компания**
3.  **Служба такси**
4.  **Оператор мобильной связи**
5.  **Массмаркет (ритейл)**
6.  **Банк**
7.  **Складской комплекс**
8.  **Научная компания (R&D)**: Работает на гранты и получает прибыль от разработок.

Каждый агент имеет уникальные параметры, функцию расчета дохода и издержек.

## Экономическая модель

### Генерация спроса

*   **Раунды 1-10**: `Demand = random(Min, Max)`
*   **Раунды 11+**: `Demand = a - b1 * P + b2 * C + b3 * A + Shock_Factor`
    *   `P` — цена игрока
    *   `C` — средняя цена конкурентов
    *   `A` — инвестиции в маркетинг/популярность
    *   `Shock_Factor` — мультипликатор от активного шока

### Шоки

Шоки — это случайные события, которые значительно влияют на параметры модели. Примеры шоков:
*   "Цены на нефть упали на 15%" → Снижается спрос и прибыль нефтяных компаний.
*   "Правительство увеличивает налоги на строительство" → Увеличиваются издержки строителей.
*   "Технологический прорыв" → Доход научной компании увеличивается на 20%.
*   "Финансовый кризис" → Повышается ставка рефинансирования.

### Расчет прибыли и капитала

Для каждого актива прибыль рассчитывается по своей формуле (например, `Прибыль = Спрос * Цена` для товаров, или `Прибыль = %_доли * Прибыль_компании` для акций).

**Итоговый капитал игрока за раунд:**
`Капитал_(n+1) = Капитал_n + Сумма(Прибыль_всех_активов) - Издержки - Налоги ± Субсидии - Платежи_по_кредитам`
