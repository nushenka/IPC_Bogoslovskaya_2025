## Краткое описание

Данный проект представляет собой **экономическую игру-симулятор**, в которой игрок управляет различными виртуальными бизнесами в условиях динамически меняющегося рынка. Модель имитирует ключевые макроэкономические процессы: инфляцию, изменение ставки рефинансирования, государственные налоги и субсидии, а также случайные рыночные шоки, влияющие на спрос и цены активов.

**Ключевая особенность**: После каждого раунда игрок получает детальную аналитику и должен самостоятельно принимать стратегические решения для максимизации прибыли, а также сам ее просчитать(тренировка задач по микроэкономике, выбирать, сколько всего производить в следующем раунде, если это неправильно просчитано, то об этом говорится в следующем раунде).

## Цель проекта

Разработать и проанализировать многоагентную модель, которая демонстрирует:
* Взаимодействие различных секторов экономики.
* Влияние макроэкономических факторов и случайных событий на микроэкономические решения агентов.
* Формирование стратегического поведения игроков на основе анализа данных (графиков спроса, котировок акций, новостной ленты).
* Процесс принятия решений в условиях неполной информации и неопределенности.

## Возможности (can be updated)

* **Управление бизнесом**: Игрок может покупать и управлять одним из нескольких типов компаний.
* **Динамическая экономика**:
  * **Инфляция**: Учитывается каждый раунд.
  * **Ставка рефинансирования**: Динамически меняется, влияя на стоимость кредитов, устанавливается цб.
  * **Налоги и субсидии**: Устанавливаются "правительством" (системой).
  * **Мировой спрос**: Влияет на уровень цен и спрос.
* **Случайные события (Шоки)**: Непредсказуемые события, которые кардинально меняют рыночную конъюнктуру.
* **Система спроса**:
  * Cпрос на каждый товар генерируется случайно в первом раунде, а затем меняется с учетом шоков и может немного колебаться вне зависимости от шоков.
* **Полная аналитика после каждого раунда**: Игрок получает сводную информацию и графики для анализа своих решений.
* **Фондовый рынок**:
  * **Биржа**: Покупка и продажа акций (долей) компаний.
  * **Драгметаллы**: Инвестирование в золото и другие металлы, цена которых волатильна.
* **Нелинейные функции издержек**: Для каждого типа бизнеса реализована своя модель затрат, генерируется для каждого игрока в первом раунде отдельно, а затем меняется в зависимости от шоков, инфляции и поставлений правительств.

## Архитектура и агенты

В симуляторе представлены следующие типы агентов-компаний:

1.  **Строительная компания**
2.  **Нефтедобывающая компания**: Монополия
3.  **Служба такси**
4.  **Оператор мобильной связи**
5.  **Массмаркет (ритейл)**
6.  **Банк**
7.  **Складской комплекс**
8.  **Завод по производству машин**
9.   **Страховая компания**

Каждый агент имеет уникальные параметры, функцию расчета дохода и издержек.

## Экономическая модель

### Генерация спроса

*   **Раунды 1**: `Demand = random`
*   **Раунды 2+**: `Demand = a - b1 * P ± Shock_Factor`
    *   `P` — цена игрока
    *   `Shock_Factor` — мультипликатор от активного шока

### Шоки

Шоки — это случайные события, которые значительно влияют на параметры модели. Примеры шоков:
*   "Цены на нефть упали на 15%" → Снижается спрос и прибыль нефтяных компаний.
*   "Правительство увеличивает налоги на строительство" → Увеличиваются издержки строителей.
*   "Технологический прорыв" → Доход научной компании увеличивается на 20%.
*   "Финансовый кризис" → Повышается ставка рефинансирования.

### Расчет прибыли и капитала

Для каждого актива прибыль рассчитывается по своей формуле (например, `Прибыль = Спрос * Цена` для товаров, или `Прибыль = %_доли * Прибыль_компании` для акций).

**Итоговый капитал игрока за раунд:**
`Капитал_(n+1) = Капитал_n + Сумма(Прибыль_всех_активов) - Издержки - Налоги ± Субсидии - Платежи_по_кредитам`

## Система аналитики и принятия решений

После завершения каждого раунда игрок получает **полную аналитическую сводку**, на основе которой должен принимать решения для максимизации прибыли в следующем раунде:

### Аналитика:

1. **График спроса и предложения**
   - Динамика спроса на товары/услуги игрока
   - Сравнение с предложением конкурентов(генерируется тоже, если компания не наделена статусом монополии)
   - Эластичность спроса по цене

2. **Финансовая отчетность**
   - Выручка и прибыль по каждому активу
   - Структура издержек (постоянные/переменные)
   - Налоговая нагрузка и субсидии

3. **Биржевая аналитика**
   - Динамика котировок акций всех компаний
   - Цены на драгоценные металлы
   - Доходность портфеля ценных бумаг

4. **Макроэкономические показатели**
   - Уровень инфляции
   - Ставка рефинансирования
   - Индекс мирового спроса для нефтяной компании и биржи

5. **Сводная информация за раунд**
   - Все произошедшие шоки и их влияние (с прошлого раунда, надо все-таки давать игроку подумать, какие шоки к чему приводят)
   - Действия конкурентов (изменение цен, инвестиции)
   - Рейтинг игроков по прибыли

###  Принимаемые решения:

На основе аналитики игрок должен самостоятельно определить оптимальную стратегию:
- **Ценовая политика**: Установить цены для максимизации выручки с учетом эластичности спроса
- **Инвестиционные решения**: Покупка/продажа акций, драгметаллов, долей в бизнесе
- **Оптимизация издержек**: Корректировка производственных процессов
- **Кредитная политика**: Взятие или погашение кредитов с учетом процентной ставки
- **Реакция на шоки**: Адаптация бизнес-модели к изменившимся рыночным условиям

**Цель игрока**: На основе анализа данных найти оптимальное решение, которое приведет к максимальной прибыли в следующем раунде.
